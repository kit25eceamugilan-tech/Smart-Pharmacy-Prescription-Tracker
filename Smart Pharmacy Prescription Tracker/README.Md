# Smart Pharmacy Prescription Tracker

A simple **stack-based prescription tracker** built in C/C++ as part of academic coursework.  
This project demonstrates the use of data structures (stack operations) in a healthcare context.

---

## 🚀 Features
- Add new prescriptions (patient, medicine, dosage, duration)
- Remove the latest prescription (stack pop)
- View the most recent prescription (stack peek)
- Display all prescriptions stored
- Menu-driven console interface
